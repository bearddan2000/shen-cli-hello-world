# shen-cli-hello-world

## Description
A POC for shen programming language.

## Tech stack
- g++
- make

## Docker stack
- ubuntu:22.04
- rzuckerm/shen

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
